Replace the remote MAC address with your ESP32 MAC address.

```
uint8_t remoteDevice[] = {0x24, 0x0a, 0xc4, 0xef, 0xaa, 0x65};
```

When you run this project on the ESP32, you will see ESP32 MAC address:
